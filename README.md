# Caesar Cipher Encryption Tool

This project implements the Caesar Cipher encryption algorithm using Python.

## Features
- Encrypt text messages
- Decrypt encrypted messages
- User input for message and shift value

## Technologies Used
- Python

## How It Works
The Caesar Cipher shifts each letter in the message by a fixed number of positions in the alphabet.

Example:
HELLO -> KHOOR (Shift = 3)
